System for Managing Students (Java OOP Project)

This is a simple Student Management System made with Java that shows how Object Oriented Programming works with Java.
With this system, you can make a student object, store information about it, and see how features like inheritance,
constructors, static variables, and overriding/overloading work.

What the project can do(how the project works)

Making objects that stand for students and have traits Name and number
Showing students' information

Using static variables to count how many students there are
Using inheritance (the Student class is an extension of the Person class), Overriding or overloading methods Using constructors

📂project structure

📂 Student-Management-System
 ├── Person.java   → Parent class (stores basic person data)
 ├── Student.java  → Child class (student-specific data + behavior)
 └── Main.java     → Entry point of the program (runs the application)

Concepts Used

Classes and Objects
Constructors (default & parameterized)
Encapsulation (private variables)
Inheritance (extends)
Method Overriding
Method Overloading
Static variables and methods
Access Modifiers (private, protected, public)
